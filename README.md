# Sistema de Recomendação de Disciplinas

Sistema desenvolvido para auxiliar na recomendação de disciplinas com base em critérios definidos.

## Objetivo
Simular um sistema de apoio à decisão, ajudando na escolha de disciplinas a partir de organização e filtragem de dados.

## Funcionalidades
- Organização de dados de disciplinas
- Filtragem com base em critérios
- Estruturação de informações para análise

## Tecnologias utilizadas
- Python / JavaScript (dependendo do seu projeto real)
- SQL (caso tenha usado)
- Lógica de programação

## Destaque técnico
- Aplicação de lógica para tomada de decisão
- Estruturação e manipulação de dados
- Organização de regras para recomendação

## Como executar
1. Clone o repositório
2. Execute o arquivo principal do projeto

## Observação
Projeto voltado para prática de lógica e organização de dados.
